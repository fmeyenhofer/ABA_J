Instructions for interactive section alignment

To align the section to the reference template proceed as follows:

  1. scroll to move along the z-axis to find the corresponding template section.
  2. with T the template can be transformed independently. Use the navigation controls to transform the template and hit T again to apply the transformation or ESC to abort.
  3. once the section and template are approximately aligned use W to warp the section to the template.
  4. repeat the previous points until you get a satisfactory match.

With Shift W you can toggle between the section display between original and warped. Additionally you can toggle the visibility the outline points of the section and the template (which are used to compute the mapping) with Shift P (see miscellaneous)

Big Data Viewer controls

Sources and display modes

The viewer has two display modes, single-source and fused. In single-source mode, only the current source is shown (there is exactly one current source at any given time.) In fused mode, all active sources are shown overlaid.

F Switch between single-source and fused mode.
1 / 2 / … / 0 Make source 1/2/…/0 the current source.
Shift 1 / Shift 2 / … / Shift 0 Make source 1/2/…/0 active or inactive. (Toggle visibility in fused mode.)
I Switch between tri-linear and nearest-neighbor interpolation.
S Show Brightness & Color dialog.
F6 Show Visibility & Grouping dialog.
G Toggle source grouping.

Navigation

Using the mouse:

Left-drag Rotate (pan and tilt) around the point where the mouse was clicked.
Right-drag or
Middle-drag		 Move in the xy-plane.
Scroll Move along the z-axis.
Meta Scroll or
Ctrl Shift Scroll		 Zoom in and out.

Using the keyboard:

X / Y / Z Select rotation axis.
← / → Rotate clockwise / counter-clockwise around the choosen rotation axis.
↑ / ↓ Zoom in / out.
, / . Move forward / backward along the z-axis.
Shift X Rotate to the zy-plane of the current source. (Look along the x-axis of the current source.)
Shift Y or Shift A Rotate to the xz-plane of the current source. (Look along the y-axis of the current source.)
Shift Z Rotate to the xy-plane of the current source. (Look along the z-axis of the current source.)

Hold Shift to rotate and browse 10x faster.
Hold Ctrl to rotate and browse 10x slower.

Bookmarks


Shift B key Set bookmark with label key.
B key Go to bookmark with label key.
O key Set current rotation to that of bookmark with label key.

Miscellaneous


T Start manual transformation. Only the current source (respectively group) is transformed. Press T again to fix the transformation relative to the other sources. Press ESC to abort.
W Map section to template.
Shift W  Toggle original/mapped
Shift P  Toggle outline points visibility
Shift U  Update input Section with current transforms
F10 Show Record Movie dialog.
F11 Save settings file (brightness, colors, groups, bookmarks, etc.)
F12 Load settings file.
